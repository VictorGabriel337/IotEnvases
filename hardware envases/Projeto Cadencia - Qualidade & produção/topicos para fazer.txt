 1º - Primeira situação - firmware

	- fazer um contador de recebimento do sensor, se caso não tiver recebendo o sinal do sensor contar o tempo que não esta recebendo

	- caso não esteja vai abrir contagem para alertar o quanto tempo não esta recebendo sinal

	- e uma contagem de quanto tempo esta recebendo o sinal

	- caso não esteja recebendo sinal, vai ter uma contagem de 3 min e abre contagem para entrar na contagem de cadencia

	- vai ser integrado com a aplicação web pra mostrar o tempo de paragem da maquina

	

 2º - segunda situação - Firmware

	- esse contador vai ser integrado com a pagina HTML e CSS pra mostrar na aplicação web quanto de garrafa já foi produzida



3º - terceira situação - Software
